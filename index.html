Frequently Asked Questions

What is your preferred tech stack for new projects?

I primarily use Kotlin with Jetpack Compose for new Android projects, following Clean Architecture principles. I also leverage Firebase for rapid backend development and GitHub Actions for CI/CD pipelines.

Do you work with international clients?

Yes, I have successfully worked with clients from the United States, Europe, and Southeast Asia. I'm comfortable with remote collaboration and different time zones.

How do you approach security in mobile apps?

Security is a core focus. I implement secure authentication, data encryption, secure API communication, and follow OWASP mobile security guidelines. I also perform regular security audits on my projects.
